- **"고객"** 또는 **"인스턴스 관리자"**: PLANKA 라이선스를 취득하고 자체 시스템에서 소프트웨어를 운영하는 조직

## 2. 계정

### 2.1 계정 생성
모든 최종 사용자는 본인 명의의 개인 계정이 필요합니다. 계정 공유는 허용되지 않습니다.

### 2.2 귀하의 책임
귀하는 계정 정보의 정확성, 접근 자격 증명의 기밀 유지, 그리고 귀하의 계정으로 발생하는 모든 활동에 대해 책임을 집니다. 계정이 무단으로 사용되고 있다고 의심되는 경우, 즉시 인스턴스 관리자에게 통보해야 합니다.

### 2.3 접근 및 접근 권한 철회
귀하의 접근 권한은 고객이 관리합니다. 고객은 언제든지 귀하의 접근을 제한하거나 철회할 수 있습니다. 고객과의 관계가 종료되는 경우(예: 고용 종료), 귀하의 접근 권한도 소멸됩니다.

## 3. 허용되는 사용

### 3.1 사용 목적
귀하는 고객이 접근 권한을 부여한 목적 범위 내에서, 귀하에게 할당된 역할과 권한 안에서만 PLANKA를 사용할 수 있습니다.

### 3.2 금지 행위
다음 행위는 명시적으로 금지됩니다:
- 접근 자격 증명을 타인과 공유하는 행위
- 불법적, 차별적 또는 학대적 목적의 사용
- 유해 파일(바이러스, 악성코드) 또는 불법 콘텐츠 업로드
- 타 사용자의 데이터에 무단으로 접근을 시도하는 행위
- 소프트웨어의 역공학, 디컴파일 또는 분해
- 고객에게 라이선스된 범위를 초과하는 사용
- 소프트웨어 또는 그 일부의 복사, 복제, 배포

### 3.3 콘텐츠에 대한 책임
귀하는 PLANKA에서 생성, 업로드 또는 공유하는 모든 콘텐츠에 대해 책임을 집니다. 해당 콘텐츠가 제3자의 권리를 침해하지 않고, 관련 법령을 준수하는지 확인해야 합니다.
